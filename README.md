# 硬件清单
- ESP32-WROOM 开发板（搭载CH340 USB转串口芯片）
- SH1106 OLED SPI 屏幕
- 若干杜邦线

### 扩展硬件
- 3P风扇


# 设备连线说明
| 开发板引脚 | 屏幕引脚 |
|------------|----------|
| GND        | GND      |
| VCC        | 3v3      |
| D22        | SCK      |
| D21        | SDA      |
| D4         | RES      |
| D13        | DC       |
| D5         | CS       |
